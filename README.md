# واحة الفيزياء - الصف العاشر

موقع تعليمي ثابت يجمع دروس واحة الفيزياء داخل تطبيق واحد دون الحاجة للخروج إلى Google Sites.

## المحتوى

- الصفحة الرئيسية.
- اشتقاق الوحدات.
- خصائص المتجهات.
- جمع المتجهات.
- التمييز بين الكميات القياسية والمتجهة.
- التمييز بين الإزاحة والمسافة.
- التحويلات.
- الحركة بتسارع ثابت.
- وصف منحنيات الحركة.
- المقذوف الرأسي.
- السقوط الحر.
- أشكال القوى.
- القوة المركزية.
- قانون نيوتن الثالث.

## الملفات

- `index.html`: هيكل الموقع.
- `styles.css`: التصميم المتجاوب.
- `script.js`: بيانات الدروس والتفاعل الداخلي.
- `assets/physics-hero.svg`: أصل بصري محلي.
- `.nojekyll`: تهيئة GitHub Pages.
